Currently this page only contains instructions on how to reproduce results from D. Panteleev, A. Ushakov - CONJUGACY SEARCH PROBLEM AND THE ANDREWS-CURTIS CONJECTURE. If you need description for some other CRAG modules, please refer to [https://github.com/stevens-crag/crag]

There is also newer and independent implementation at [https://github.com/dpantele/acc], but currently it completely lacks any documentation and does not have all routins reimplemented there.
